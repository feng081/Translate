使用流程

下载、配置、导入、翻译，四步完成论文翻译。

1

下载软件

进入 Releases 页面下载 Windows 压缩包，解压后双击运行 exe。

2

配置 API

进入设置页面，填写 API Key、Base URL、模型名称等参数。

3

导入论文

选择本地 PDF，或输入 arXiv 链接 / 论文编号进行解析。

4

查看译文

翻译完成后打开论文文件夹，查看原文 PDF 与译文 PDF。

常见问题

使用前可以先了解这些注意事项。

需要配置 API Key 吗？

需要。软件通过大模型 API 进行翻译，首次使用前需要在设置中填写 API Key 和模型信息。

译文保存在哪里？

默认保存在 C:\Users\当前用户\translate，也可以在设置中的导出目录里修改。

支持多个任务同时翻译吗？

支持。PDF 和 arXiv 任务会统一显示在左侧正在翻译列表中，可以点击切换查看进度。

暂停和停止会立即生效吗？

如果任务正在等待一次 API 请求返回，暂停或停止会在当前请求结束后的安全点生效。
